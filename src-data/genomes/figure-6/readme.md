# ecoli-genomes-fig6

Set of E. coli genomes to be used for the distance matrix and dendrogram in figure 6.

## Files

* `genomes.csv`: Table of genome IDs and their MLST and phylotypes.
* `fasta/`: Contains sequences in FASTA format.
